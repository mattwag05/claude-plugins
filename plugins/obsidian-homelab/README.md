# obsidian-homelab

Personal Obsidian vault intelligence, homelab operations, and daily workflow skills for Matthew Wagner's Claude Code setup.

## Skills

| Skill | Trigger phrases | Purpose |
|-------|----------------|---------|
| `obsidian-markdown` | write note, format note, obsidian markdown | OFM syntax + Matt's Vault conventions |
| `obsidian-cli` | obsidian CLI, vault command, read note | CLI commands for both vaults |
| `defuddle` | extract article, parse page, web clip | Clean web page content extraction |
| `vault-navigator` | find notes, search vault, where does this go | Navigate and audit Matt's Vault structure |
| `polaris` | top of mind, goals, priorities, what should I work on | AI navigation anchor — read before making recommendations |
| `homelab-ops` | docker, pihole, talia, homelab, service health | Day-to-day homelab operations |
| `daily-workflows` | morning briefing, daily, project status, beads | Morning briefing and daily workflow orchestration |
| `ccl-manager` | context file, CCL, Claude Context Library | Manage CCL context documents |

## Vault Structure

Matt uses two Obsidian vaults:
- **Matt's Vault** — personal, medical, project notes (~1,314 files)
- **Claude Context Library (CCL)** — technical references for Claude (~280 files)

## Installation

This plugin is part of the `matthewwagner/claude-plugins` Forgejo marketplace.
